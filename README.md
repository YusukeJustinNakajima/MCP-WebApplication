# MCP-WebApplication

TypeScript Express app with modern tooling.

## Quick Start

```bash
npm install
npm run dev
```

## Features

- TypeScript + Express
- CORS enabled  
- Enhanced logging (helper-logger)
- HTTP client (Axios)

## API

- `GET /` - Welcome page
- `GET /quote` - Random quote

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Compile TypeScript  
- `npm start` - Run production server
