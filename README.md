# MCP-WebApplication

Minimal TypeScript Express app with logging, CORS, and API integration.

## Features

- Express + TypeScript
- Logging with Morgan & helper-logger
- CORS support
- Environment config
- HTTP client (Axios)

## Setup

```bash
npm install
npm run dev
```

## API

- `GET /` - Hello page
- `GET /quote` - Random quote

## Scripts

- `dev` - Development server
- `build` - Compile TypeScript
- `start` - Production server
