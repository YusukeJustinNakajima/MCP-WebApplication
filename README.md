# MCP-WebApplication

Minimal TypeScript Express app with logging, CORS, and API integration.

## Quick Start

```bash
npm install
npm run dev
```

## Features

- Express + TypeScript  
- CORS support
- Environment config
- HTTP client (Axios)

## Endpoints

- `GET /` - Hello page
- `GET /quote` - Random quote

## Scripts

- `npm run dev` - Development server
- `npm run build` - Compile TypeScript  
- `npm start` - Production server
